## Neon Maze

Neon Maze is a fast-paced survival maze game set in a glowing cyberpunk grid where visibility is your greatest enemy.

Navigate the maze using WASD or Arrow Keys while surviving periodic blackouts, shifting walls, and increasingly aggressive maze behavior.

### Features

- Blackout System
  - The screen periodically flickers into darkness
  - A shrinking flashlight centered on the player is the only source of visibility
  - Blackout frequency increases as levels progress
- Dynamic Maze Shifting
  - Walls randomly open and close during gameplay
  - Maze mutations become more intense at higher levels
  - Forces players to constantly adapt their path
- Progressive Difficulty
  - Each level starts with a time limit
  - Initial timer: 2 minutes
  - Every level reduces available time by 10 seconds
  - Higher levels increase:
    - blackout frequency
    - wall-shift intensity
    - visibility pressure
